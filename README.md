# TestProj-DBTreeViewManipulator

Решение тестового задания для претендентов на должность инженера-программиста 

## Описание задачи

### Дано:

В базе данных представлены три таблицы:

1. Таблица "Связи" (Links): содержит информацию о связях между объектами с полями idparent, idchild и linkname.
2. Таблица "Объекты" (Objects): хранит объекты с полями id, type и product.
3. Таблица "Атрибуты" (Attributes): содержит атрибуты объектов с полями id, name и value.

### Необходимо:

Разработать программный модуль, который выполняет следующие функции:

1. Отображение в компоненте типа TreeView данных из указанной выше БД;
2. Добавление и удаление элементов;
3. Сохранение изменений обратно в БД;
4. Выгрузка структуры БД в файл формата XML.

## Требования

1. СУБД: Microsoft SQL Server.
2. Язык реализации: C#.
